# Create Plan Command

You just explored the problem deeply. Now create a detailed implementation plan.

Format (markdown file output):
```
# Plan: [Feature Name]

## TLDR
1-2 sentence summary of what we're building and why.

## Critical Decisions
- [Decision 1]: Why we chose this
- [Decision 2]: Why we chose this
- [Decision 3]: Why we chose this

## Implementation Steps
Each step format:
- [ ] Step 1: [Clear, specific action]
- [ ] Step 2: [Clear, specific action]
- [ ] Step 3: [Clear, specific action]

Add 10-15 steps total. Break it down into chunks that take 15-30 min each.

## Potential Issues
- Risk 1 and mitigation
- Risk 2 and mitigation
```

Make the plan so detailed that someone could follow it step-by-step.
